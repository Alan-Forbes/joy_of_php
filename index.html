Introduction

The Joy of PHP book walks you through creating a PHP-powered web application for Sam's Used Cars.  All the sample code in the book is conveniently created here.  To get started, you'll need to make sure that you have PHP and mySQL running on your machine.  Here are the links to get started. 

Getting Started

Before you can do much of anything you need to install PHP, Apache, and mySQL. The following packages do it automatically.

Windows users: Download WAMP

Mac Users: Download MAMP (Pro version not required)

Free Code Editors

** Visual Studio Code is my recommended editor **

Programmer's Notepad is a good all purpose editor

Zed is an up and coming editor with a lot of promise.
